## 리서치 결과
- 비슷한 앱[1]: 트리플
- 장점 : 사용자가 수동으로 동선,장소를 입력하여 보여주는 여행 앱
- 차별점 : 수동으로 정보를 입력하는 것이 아닌 백그라운드에서 사용자를 추적하는 앱을 만들 것

- 비슷한 앱[2]: 트래블리맵
- 여행 동선짜기 전용어플의 느낌이 더 강하다

- 비슷한 앱[3]: 여다

- 비슷한 앱[3]: 포토로그

- 

- 비슷한 앱[3]: 무다
